&#x20; - question: What does DNS stand for?

&#x20;   options:

&#x20;     - Domain Name System

&#x20;     - Dynamic Network Service

&#x20;     - Data Node Server

&#x20;     - Distributed Name System

&#x20;   correct: Domain Name System

&#x20;   explanation: DNS translates domain names like google.com into IP addresses.

&#x20; - question: Which port does HTTPS use?

&#x20;   options:

&#x20;     - "80"

&#x20;     - "443"

&#x20;     - "21"

&#x20;     - "8080"

&#x20;   correct: "443"

&#x20;   explanation: HTTPS uses port 443 for secure encrypted communication.

&#x20; - question: What does IP stand for?

&#x20;   options:

&#x20;     - Internet Protocol

&#x20;     - Internal Process

&#x20;     - Input Port

&#x20;     - Interface Provider

&#x20;   correct: Internet Protocol

&#x20;   explanation: IP is the principal communications protocol for routing packets across networks.
